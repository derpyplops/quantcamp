# Quantcamp 

This is the website for Trading Bootcamp by Ricki Heicklen.

## Setup

### Locally

1. Clone the repo
2. `pnpm install`
3. `pnpm dev`

### Deploying

4. `pnpm build`
5. `surge --project ./dist --domain quantcamp.surge.sh` (*)

(*) You'll need to have surge installed and logged in, and to be added as a collaborator.
